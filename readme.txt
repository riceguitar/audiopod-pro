Sermon Manager Pro is an addon to Sermon Manager, the most popular sermon plugin for WordPress. It includes templates, page builder support, page assignment, and much more.

== Description ==

Sermon Manager Pro enhances the functionality of the Sermon Manager plugin by adding additional features, templates, and integrations. Enjoy seamless support with page builders like Divi and Elementor, page assignment, and customized sermon templates.

**Features Include:**
* Change your look with Templates
* Multiple Podcast Support
* Divi Support & Custom Divi Builder Modules
* Custom Elementor Elements
* Custom Beaver Builder Modules
* Custom WPBakery Page Builder Modules
* Works with YOUR theme
* Page Assignment for Archive & Taxonomy
* Migration from other plugins is a breeze
* SEO & Marketing Ready
* Live Chat Support Inside the Plugin
* PowerPress Compatibility
* [Full List of Pro Features](https://sermonmanager.pro/features)

== Installation ==

1. Upload the `sermon-manager-pro` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the settings under the Sermon Manager Pro tab in the WordPress admin.

== Faq ==

= Do I need Sermon Manager to use this plugin? =

Yes, Sermon Manager Pro is an addon for Sermon Manager, so you must have the Sermon Manager plugin installed.

= Does this plugin work with page builders? =

Yes, Sermon Manager Pro works with popular page builders like Divi and Elementor.
